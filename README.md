## 🧪 How I Built My First LLM From Scratch — Zero Dollars

> A complete end-to-end journey from zero knowledge to a live deployed AI model

```
Step 1 — Data Collection
Collected 40+ professional resumes across DevOps MLOps Cloud LLMOps roles
Formatted as clean plain text training data
Saved as resume_data.txt — 100KB of high quality training data

Step 2 — Environment Setup
Created free Kaggle account with GPU T4 x2
30 GPU hours per week — completely free
Cloned nanoGPT from Andrej Karpathy's GitHub

Step 3 — Data Preparation
Tokenized resume text using tiktoken GPT-2 BPE encoding
Split into 90% training and 10% validation sets
Saved as binary files for efficient loading

Step 4 — Model Training
Trained 10M parameter transformer on free Kaggle GPU
Loss dropped from 4.26 to 1.42 over training run
Saved checkpoints automatically to Google Drive

Step 5 — Fine-Tuning with LoRA
Applied LoRA adapters to GPT-2 117M parameter model
Trained on resume instruction-response pairs
Only 2M parameters trained — 98% fewer than full fine-tuning

Step 6 — Deployment
Uploaded model to HuggingFace Hub
Built Gradio demo interface
Deployed as HuggingFace Space — live for anyone to use

Step 7 — Result
Live demo at huggingface.co/spaces/GopiKrishna88/resume-ai-demo
Zero dollars spent throughout entire process
Complete MLOps pipeline — data to production
```
